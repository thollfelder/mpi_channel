int channel_receive_pt2pt_spsc_sync ( MPI_Channel *  ch,
void *  data 
)

Receives the numbers of bytes of a data element specified in channel_alloc() from the channel and stores them starting at the adress the void pointer holds. channel_receive_pt2pt_spsc_sync() blocks until a matching channel_send_pt2pt_spsc_sync() is called.

Parameters
[in]chPointer to a MPI_Channel of type PT2PT SPSC SYNC
[in]dataPointer to a memory adress of which size bytes will be received to
Returns
Returns 1 if receiving was successful, -1 otherwise
Note
Returns -1 if internal problems with MPI_Recv() happend
